轻笔记应用数据收集与使用说明

1. 数据收集概述

轻笔记应用遵循"最小化收集"原则，仅收集用户明确提供且必要的数据，不进行任何不必要的数据收集。

1.1 主动提供的数据

数据类型 收集方式 存储方式 用途
笔记内容 用户输入 本地加密存储 用于展示和编辑笔记
笔记图片 用户从相册导入 本地文件系统存储 用于笔记中的图片展示
操作记录 用户创建、编辑、删除笔记 本地加密存储 用于维护笔记的创建时间、更新时间和状态

1.2 自动收集的数据

轻笔记应用不自动收集任何用户数据，包括但不限于：

  • 位置信息
  • 设备标识符
  • 网络活动
  • 使用习惯
  • 崩溃日志

2. 数据存储与安全

2.1 存储方式

  • 数据存储位置：所有用户数据均存储在设备本地，不上传至任何服务器
  • 存储形式：
    • 文本数据：使用JSON格式+XOR加密算法+Base64编码存储
    • 图片数据：原始图片文件存储在设备本地文件系统

2.2 加密机制

为确保用户数据安全，应用采用了以下加密措施：

  1. 对称加密算法：使用XOR位运算结合自定义密钥进行数据加密
  2. 密钥管理：加密密钥内置在应用代码中，确保每次加密/解密操作的一致性
  3. Base64编码：加密后的文本通过Base64编码，确保特殊字符正确存储
  4. 完整覆盖：所有存储的笔记数据和回收站数据均经过加密处理

2.3 加密实现细节

// 加密函数示例
function encrypt(text) {
  let result = '';
  const SECRET_KEY = 'easynote_secure_key_2023';
  for (let i = 0; i < text.length; i++) {
    const charCode = text.charCodeAt(i) ^ SECRET_KEY.charCodeAt(i % SECRET_KEY.length);
    result += String.fromCharCode(charCode);
  }
  return btoa(unescape(encodeURIComponent(result)));
}

// 解密函数示例
function decrypt(encryptedText) {
  try {
    const text = decodeURIComponent(escape(atob(encryptedText)));
    let result = '';
    const SECRET_KEY = 'easynote_secure_key_2023';
    for (let i = 0; i < text.length; i++) {
      const charCode = text.charCodeAt(i) ^ SECRET_KEY.charCodeAt(i % SECRET_KEY.length);
      result += String.fromCharCode(charCode);
    }
    return result;
  } catch (error) {
    console.error('解密失败:', error);
    return '';
  }
}

3. 数据使用目的

用户提供的数据仅用于以下目的：

  1. 笔记功能实现：存储和展示用户创建的笔记内容
  2. 用户体验优化：维护笔记的创建时间、更新时间等元数据
  3. 数据管理：支持笔记的编辑、删除、恢复等操作

4. 数据共享与披露

轻笔记应用不共享或披露任何用户数据：

  • 不与第三方服务共享数据
  • 不向开发者传输数据
  • 不用于广告或营销目的
  • 不进行数据分析或统计

5. 用户权利

作为应用用户，您拥有以下权利：

  1. 完全控制权：您可以随时查看、编辑、删除或恢复自己的笔记
  2. 数据删除权：您可以永久删除任何笔记数据
  3. 隐私保护：您的数据永远不会被应用开发者或第三方访问
  4. 数据迁移：您可以通过备份设备数据的方式迁移应用数据

6. 数据保留期限

  • 正常笔记：无限期保留，直到用户主动删除
  • 回收站笔记：无限期保留，直到用户主动恢复或永久删除
  • 永久删除：数据将被立即从设备中移除

7. 隐私政策更新

本数据收集与使用说明可能会根据应用功能的变化而更新。更新后的说明将在应用内发布，并自发布之日起生效。
